### Scripts
- Install: ```npm install```
- Build: ```npm run build```
- Lint: ```npm run lint:fix```
- Pretty: ```npm run prettier:fix```
- Run unit tests: ```npm run test```
- Start server: ```npm run start```

### Usage
The server will listen on port 3000:

#### instructions:open the browser and listen http://localhost:3000/
